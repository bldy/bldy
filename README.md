**bldy** is a concurrent build system that runs blazingly fast, incrementally and in parallel.

## Getting started

### Requirements

* go (+1.12)
  * Mac: `brew install golang`
  * ArchLinux: `pacman -S golang`

### Install:

```
go get bldy.build/bldy/src/cmd/bldy
```
